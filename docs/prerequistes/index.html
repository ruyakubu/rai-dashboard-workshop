Prerequisites

Azure subscription account#

In order to successfully complete this workshop you'll need Azure credits. The instructor will provide you with your account access code.

Install and Setup your cloud resources#

We'll be using Azure Cloud Shell to automate and expedite creating your Azure Machine Learning workspace; as well as running the jobs to train and register your model. You'll be using a bash script. And finally, created the Responsible AI dashboard that you'll be using in this workshop.

  1. Log into the Azure portal with your account subscription

  1. Select the "Azure Cloud Shell" icon.

A command terminall will open at the bottom of the page.

  1. In Azure Cloud Shell, clone the github repository
git clone https://github.com/ruyakubu/RAI-Diabetes-Hospital-Readmission-classification.git
  1. Change to the project directory
cd RAI-Diabetes-Hospital-Readmission-classification

Azure Login#

  1. At the Terminal command prompt, enter the following Azure CLI command to log into your Azure account. Open the link provided and enter access code
az login --use-device-code
  1. Once you have successfully logged in, set your account to the the current subscription.
az account set -s <enter-your-subscription-id-here>

Create Resource Group and Azure ML workspace#

  1. Create an Azure resource group. For location, enter westus, westus2, eastus or westeurope,
az group create --name <resource-group-name> --location <location>
  1. Create an Azure Machine Learning workspace using the resource group name you created.
az ml workspace create -n <workspace-name> -g <resource-group-name>
  1. Set your environment default to the resource group and Azure ML workspace.
az configure --defaults group="<resource-group-name>" workspace="<workspace-name>"

Run jobs for training the model and creating the RAI dashboard#

  1. Run the following bash script that will create a compute instance; register the train/test datasets; train and register the model; then create the RAI dashboard for you.
bash cloud/setup.sh
  1. After the setup script has successfully completed, log into Azure Machine Learning studio to monitor the pipeline job for creating the RAI dashboard. Then, click on the Pipelines tab to get the job status.

  1. To view the progression of the pipeline job creating the RAI dashboard, click on the job name.

  1. Terrific...you're ready to start using the dashboard!

  2. Click on the "Next" button below to proceed to the Error Analysis lab.
